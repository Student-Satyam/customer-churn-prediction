# Customer Churn Analysis and Prediction

## Project Overview

This project focuses on analyzing customer churn in a telecommunications company and building a machine learning model to predict whether a customer is likely to leave the service.

Customer churn prediction helps businesses identify at-risk customers and improve customer retention strategies.

---

## Objectives

- Perform data preprocessing and cleaning
- Handle missing values
- Encode categorical variables
- Split data into training and testing sets
- Train a machine learning classification model
- Evaluate model performance using multiple metrics
